Выполненная интеграция включает:
1. Создание HTML-формы для ввода информации о кредитной карте с привязкой элементов ввода к библиотеке Stripe JS
для безопасного сбора данных карты на клиентской стороне.

2. Использование PHP-скрипта для создания PaymentIntent с поддержкой 3D Secure
и сохранения информации о транзакции в базе данных, используя библиотеку Stripe PHP.

3. Получение PaymentIntent и информации о клиенте с помощью другого PHP-скрипта с использованием Stripe API.

Таким образом, у пользователя есть возможность ввести данные карты через защищенную форму,
после чего происходит создание PaymentIntent через сервер и информация о платеже сохраняется в базе данных.
De facto, это обеспечивает интеграцию платежной системы в веб-приложение, поддерживая не только обработку данных карт,
но и сохранение информации о платежах для последующей обработки.
